# AI Voice-Enabled Browser Extension for Accessible Web Interaction

## Project Overview
This extension allows users to control the browser using voice commands. It converts speech to text, detects user intent using NLP, and performs actions like opening websites and navigating pages.

## Features
- Voice command recognition
- Hands-free browser control
- Open websites using voice
- Navigate pages
- Accessible interaction

## Technologies Used
- JavaScript
- HTML
- CSS
- Web Speech API
- Browser Extension APIs

## Installation
1. Download the project
2. Open Chrome
3. Go to chrome://extensions/
4. Enable Developer Mode
5. Click Load unpacked
6. Select the project folder
